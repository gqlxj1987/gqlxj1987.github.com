read-write quorum system

#databend

原文链接

区分了read-write quorum system以及majority系统之间的区别

针对

  • 加入节点 d，即从 {a,b,c} 变成 {a,b,c,d}。
  • 删除节点 a，即从 {a,b,c,d} 变成 {b,c,d}。

的情况，针对其中的网络隔离的情况，read-write符合替换，但majority达不到要求

join consensus算法

M(abc) x M(bcd) = {
    ab ∪ bc,
    ab ∪ cd,
    ab ∪ bd,
    bc ∪ bc,
    bc ∪ cd,
    bc ∪ bd,
    ac ∪ bc,
    ac ∪ cd,
    ac ∪ bd,
} = {
    abc,
    abcd,
    abd,
    acd,
    bc,
    bcd,
} = {M(a,b,c,d),{b,c}}
