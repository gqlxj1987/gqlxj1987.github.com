One of the most important aspects of building any supervised learning model on numeric data is to understand the features well. Looking at partial dependence plots of a model helps you understand how the modelвҖҷs output changes with any feature.

  1. Feature understanding
  2. Identifying noisy features (the most interesting part!)
  3. Feature engineering
  4. Feature importance
  5. Feature debugging
  6. Leakage detection and understanding
  7. Model monitoring

We can see that higher the trend-correlation threshold to drop features, higher is the leaderboard (LB) AUC.

  1. Checking if the featureвҖҷs population distribution looks right. IвҖҷve personally encountered extreme cases like above numerous times due to minor bugs.
  2. Always hypothesize what the feature trend will look like before looking at these plots. Feature trend not looking like what you expected might hint towards some problem. And frankly, this process of hypothesizing trends makes building ML models much more fun!

жҜ«ж— ж„ҸеӨ–пјҢй’ұиҝҷдёӘдёңиҘҝжҳҜдёӘжӯ»з»“пјҢдёҖи°ҲеҲ°й’ұзҡ„й—®йўҳпјҢжһңз„¶иҝҳжҳҜзҶҹжӮүзҡ„еңәжҷҜгҖӮеҰӮжһңпјҢжІЎжңүеҸ‘з”ҹйӮЈд»¶дәӢжғ…е°ұеҘҪдәҶпјҢзҺ°еңЁд№ҹдёҚдјҡжүҜеҮәжқҘиҝҷд№ҲеӨҡпјҢдёҚдјҡз”ЁеҸҰдёҖдёӘй”ҷиҜҜжқҘжҺ©зӣ–иҝҷж ·дёҖдёӘй”ҷиҜҜпјҢй”ҷдёҠеҠ й”ҷгҖӮгҖӮгҖӮе“ҺгҖӮгҖӮгҖӮиҝҷдёӘжҳҹжңҹиҒҠдёҖдёӢпјҢз»ҷдёӘдҝЎжҒҜеҮәжқҘгҖӮгҖӮгҖӮ

зӣёжҜ”иҫғиҖҢиЁҖпјҢиҮӘе·ұиҝҳжҙ»зқҖпјҢиҝҳз®—е№ёиҝҗдәҶпјҢдҪҶд№ҹжҳҜдёҚе№ёзҡ„пјҢдёҚиҜҘзүөжүҜеҲ°еҲ«дәәеҲ°иҮӘе·ұзҡ„з”ҹжҙ»пјҢж— жі•з»ҷеҜ№ж–№жғіиҰҒзҡ„з”ҹжҙ»пјҢеҲ«дәәжҳҺжҳҺжңүжӣҙеҘҪзҡ„жңәдјҡеҺ»иҺ·еҸ–жӣҙеҘҪзҡ„з”ҹжҙ»пјҢе…¶е®һпјҢдёҖдёӘдәәеӯӨзӢ¬ең°иҖҒеҺ»пјҢиҝҷж ·пјҢдёҚз”ЁеёҰз»ҷд»–дәәзғҰжҒјпјҢдёҖдәәеҗғйҘұпјҢе…Ёе®¶дёҚж„ҒгҖӮе®үе®үеҝғеҝғең°еҒҡеҘҪиҮӘе·ұеә”иҜҘеҒҡзҡ„пјҢйқҷеҝғең°з…§йЎҫеҘҪе°ұиЎҢпјҢд№ҹеҸӘиғҪе°ҪиҮӘе·ұеҠӘеҠӣдәҶпјҢжҜ•з«ҹе…¶д»–ж–№йқўпјҢиҮӘе·ұд№ҹз»ҷдёҚдәҶгҖӮгҖӮиӢҰз¬‘гҖӮгҖӮ
