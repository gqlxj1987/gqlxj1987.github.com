Love will set you free

发表于 2016-11-15   |   分类于 love   |  

当你年轻的时候，你永远不会知道你的生涯在何时结束，所以，你需要制定一些小目标，一步一个台阶，一步一步地去实现。 ——致敬K神

正如题目所说，love will set you free. 如果你所想要的爱情，不让你感觉那么free，那这份感情是否是你所需要的，你需要好好想一想。如果，如何去维护这份感情，让你感觉很痛苦的话，是否是该尝试放弃一些试试，不要抓的那么紧。。

周末去了躺济南，交出了自己的第一次牵手，以及初吻，差点的初次。一切感觉地那么不真实，最后的结论还是没有答案。其实，已经有答案了，不是？不是炮友的炮友关系。。哈哈。。

自己还是要求的太多了，妄图用自己看起来比较重的感情来框住人，其实一切都是自己的自以为是。其实，这点上也要检讨一下自己，还是太年轻，妄图用自己的价值观来改变他人，其实，就算改变了，人也变成另外一个样子，有什么意义。其实，这也是给自己的固执买单，自己的自以为是。。呵呵。。

自己还是做不到奇怪感觉下的一些行为，虽然尝试性地去说些话，但感觉还是偏离了自己的想法，如果要靠这些动作和语言来实现自己的目的，那么之前为什么要开启这段，把自己变得不像自己。。。

奇怪的感情，奇怪的感觉，奇怪的世界。。。还是理解不了，乖乖地当好自己的acer。

后面，还是慢慢放开心态，做到平常心，像普通的朋友，不发除了扇子脸的其他表情，做到自律，克已，慢慢地将这段感情转化为正常的友情，努力地让自己free吧。。

同上篇所说，初恋，尚未开始，就已经结束。。也教会了自己很多的东西。。

这周开始制定时间表，慢慢恢复自律的时间表，努力地去提高自己，人家期望的是公主和王子的剧情。。虽然跟王子扯不上边，但能做个暴发户也不错，加油做个暴发户
