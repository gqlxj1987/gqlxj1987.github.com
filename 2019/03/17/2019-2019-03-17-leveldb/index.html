еҺҹж–Үй“ҫжҺҘ

жҲ‘д»¬еҸҜд»Ҙд»ҺдёӨдёӘдёҚеҗҢзҡ„и§’еәҰжқҘзңӢеҫ…LevleDbпјҡйқҷжҖҒи§’еәҰе’ҢеҠЁжҖҒи§’еәҰгҖӮ

  • д»ҺйқҷжҖҒи§’еәҰпјҢеҸҜд»ҘеҒҮжғіж•ҙдёӘзі»з»ҹжӯЈеңЁиҝҗиЎҢиҝҮзЁӢдёӯпјҲдёҚж–ӯжҸ’е…ҘеҲ йҷӨиҜ»еҸ–ж•°жҚ®пјүпјҢжӯӨж—¶жҲ‘д»¬з»ҷLevelDbз…§зӣёпјҢд»Һз…§зүҮеҸҜд»ҘзңӢеҲ°д№ӢеүҚзі»з»ҹзҡ„ж•°жҚ®еңЁеҶ…еӯҳе’ҢзЈҒзӣҳдёӯжҳҜеҰӮдҪ•еҲҶеёғзҡ„пјҢеӨ„дәҺд»Җд№ҲзҠ¶жҖҒзӯүпјӣ

  • д»ҺеҠЁжҖҒзҡ„и§’еәҰпјҢдё»иҰҒжҳҜдәҶи§Јзі»з»ҹжҳҜеҰӮдҪ•еҶҷе…ҘдёҖжқЎи®°еҪ•пјҢиҜ»еҮәдёҖжқЎи®°еҪ•пјҢеҲ йҷӨдёҖжқЎи®°еҪ•зҡ„пјҢеҗҢж—¶д№ҹеҢ…жӢ¬йҷӨдәҶиҝҷдәӣжҺҘеҸЈж“ҚдҪңеӨ–зҡ„еҶ…йғЁж“ҚдҪңжҜ”еҰӮcompactionпјҢзі»з»ҹиҝҗиЎҢж—¶еҙ©жәғеҗҺеҰӮдҪ•жҒўеӨҚзі»з»ҹзӯүзӯүж–№йқўгҖӮ

еҜ№дәҺеҲҶжһҗдёҖдёӘж•°жҚ®еә“пјҢд№ҹеҹәжң¬йҒөеҫӘиҝҷж ·зҡ„жҖқи·Ҝ

жһ„жҲҗLevelDbйқҷжҖҒз»“жһ„зҡ„еҢ…жӢ¬е…ӯдёӘдё»иҰҒйғЁеҲҶпјҡеҶ…еӯҳдёӯзҡ„MemTableе’ҢImmutable MemTableд»ҘеҸҠзЈҒзӣҳдёҠзҡ„еҮ з§Қдё»иҰҒж–Үд»¶пјҡCurrentж–Үд»¶пјҢManifestж–Үд»¶пјҢlogж–Үд»¶д»ҘеҸҠSSTableж–Үд»¶гҖӮ

еҪ“еә”з”ЁеҶҷе…ҘдёҖжқЎKey:Valueи®°еҪ•зҡ„ж—¶еҖҷпјҢLevelDbдјҡе…ҲеҫҖlogж–Үд»¶йҮҢеҶҷе…ҘпјҢжҲҗеҠҹеҗҺе°Ҷи®°еҪ•жҸ’иҝӣMemtableдёӯпјҢиҝҷж ·еҹәжң¬е°ұз®—е®ҢжҲҗдәҶеҶҷе…Ҙж“ҚдҪңпјҢеӣ дёәдёҖж¬ЎеҶҷе…Ҙж“ҚдҪңеҸӘж¶үеҸҠдёҖж¬ЎзЈҒзӣҳйЎәеәҸеҶҷе’ҢдёҖж¬ЎеҶ…еӯҳеҶҷе…ҘпјҢжүҖд»ҘиҝҷжҳҜдёәдҪ•иҜҙLevelDbеҶҷе…ҘйҖҹеәҰжһҒеҝ«зҡ„дё»иҰҒеҺҹеӣ 

logзҡ„дҪңз”Ёдё»иҰҒжҳҜз”ЁдәҺзі»з»ҹеҙ©жәғжҒўеӨҚ

жңүеәҸзҡ„keyеҖјпјҢеҜ№дәҺе…¶е®ғLevelзҡ„SSTableж–Үд»¶жқҘиҜҙпјҢеҲҷдёҚдјҡеҮәзҺ°еҗҢдёҖеұӮзә§еҶ….sstж–Үд»¶зҡ„keyйҮҚеҸ зҺ°иұЎпјҢе°ұжҳҜиҜҙLevel Lдёӯд»»ж„ҸдёӨдёӘ.sstж–Үд»¶пјҢйӮЈд№ҲеҸҜд»ҘдҝқиҜҒе®ғд»¬зҡ„keyеҖјжҳҜдёҚдјҡйҮҚеҸ зҡ„

з”ұдәҺlogйҮҮз”Ё32KдҪңдёәдёҖдёӘblockйғЁеҲҶпјҢеҝ…з„¶еҮәзҺ°ж•°жҚ®жӢҶеҲҶзҡ„й—®йўҳпјҢиҝҷж—¶еҖҷпјҢж•°жҚ®еҲҶдёәеҮ дёӘзұ»еһӢпјҡFULL/FIRST/MIDDLE/LAST

SSTable

еҸҜд»Ҙе°Ҷ.sstж–Үд»¶еҲ’еҲҶдёәж•°жҚ®еӯҳеӮЁеҢәе’Ңж•°жҚ®з®ЎзҗҶеҢәпјҢж•°жҚ®еӯҳеӮЁеҢәеӯҳж”ҫе®һйҷ…зҡ„Key:Valueж•°жҚ®пјҢж•°жҚ®з®ЎзҗҶеҢәеҲҷжҸҗдҫӣдёҖдәӣзҙўеј•жҢҮй’Ҳзӯүз®ЎзҗҶж•°жҚ®пјҢзӣ®зҡ„жҳҜжӣҙеҝ«йҖҹдҫҝжҚ·зҡ„жҹҘжүҫзӣёеә”зҡ„и®°еҪ•гҖӮдёӨдёӘеҢәеҹҹйғҪжҳҜеңЁдёҠиҝ°зҡ„еҲҶеқ—еҹәзЎҖдёҠзҡ„пјҢе°ұжҳҜиҜҙж–Үд»¶зҡ„еүҚйқўиӢҘе№Іеқ—е®һйҷ…еӯҳеӮЁKVж•°жҚ®пјҢеҗҺйқўж•°жҚ®з®ЎзҗҶеҢәеӯҳеӮЁз®ЎзҗҶж•°жҚ®гҖӮз®ЎзҗҶж•°жҚ®еҸҲеҲҶдёәеӣӣз§ҚдёҚеҗҢзұ»еһӢпјҡзҙ«иүІзҡ„Meta BlockпјҢзәўиүІзҡ„MetaBlock зҙўеј•е’Ңи“қиүІзҡ„ж•°жҚ®зҙўеј•еқ—д»ҘеҸҠдёҖдёӘж–Үд»¶е°ҫйғЁеқ—

Data Block еҶ…зҡ„ KV и®°еҪ•жҳҜжҢүз…§ Key з”ұе°ҸеҲ°еӨ§жҺ’еҲ—зҡ„пјҢж•°жҚ®зҙўеј•еҢәзҡ„жҜҸжқЎи®°еҪ•жҳҜеҜ№жҹҗдёӘ Data Block е»әз«Ӣзҡ„зҙўеј•дҝЎжҒҜпјҢжҜҸжқЎзҙўеј•дҝЎжҒҜеҢ…еҗ«дёүдёӘеҶ…е®№пјҢд»Ҙеӣҫ жүҖзӨәзҡ„ж•°жҚ®еқ— i зҡ„зҙўеј• Index i жқҘиҜҙпјҡзәўиүІйғЁеҲҶзҡ„з¬¬дёҖдёӘеӯ—ж®өи®°иҪҪеӨ§дәҺзӯүдәҺж•°жҚ®еқ— i дёӯжңҖеӨ§зҡ„ Key еҖјзҡ„йӮЈдёӘ KeyпјҢз¬¬дәҢдёӘеӯ—ж®өжҢҮеҮәж•°жҚ®еқ— i еңЁ.sst ж–Үд»¶дёӯзҡ„иө·е§ӢдҪҚзҪ®пјҢз¬¬дёүдёӘеӯ—ж®өжҢҮеҮә Data Block i зҡ„еӨ§е°ҸпјҲжңүж—¶еҖҷжҳҜжңүж•°жҚ®еҺӢзј©зҡ„пјүгҖӮеҗҺйқўдёӨдёӘеӯ—ж®өеҘҪзҗҶи§ЈпјҢжҳҜз”ЁдәҺе®ҡдҪҚж•°жҚ®еқ—еңЁж–Үд»¶дёӯзҡ„дҪҚзҪ®зҡ„

е…ідәҺж•°жҚ®и®°еҪ•пјҢиҝҳжңүйҮҚеҗҜзӮ№

вҖңйҮҚеҗҜзӮ№вҖқ зҡ„ж„ҸжҖқжҳҜпјҡеңЁиҝҷжқЎи®°еҪ•ејҖе§ӢпјҢдёҚеҶҚйҮҮеҸ–еҸӘи®°иҪҪдёҚеҗҢзҡ„ Key йғЁеҲҶпјҢиҖҢжҳҜйҮҚж–°и®°еҪ•жүҖжңүзҡ„ Key еҖјпјҢеҒҮи®ҫ Key i+1 жҳҜдёҖдёӘйҮҚеҗҜзӮ№пјҢйӮЈд№Ҳ Key йҮҢйқўдјҡе®Ңж•ҙеӯҳеӮЁ вҖңthe colorвҖқпјҢиҖҢдёҚжҳҜйҮҮз”Ёз®Җз•Ҙзҡ„ вҖңolorвҖқ ж–№ејҸгҖӮBlock е°ҫйғЁе°ұжҳҜжҢҮеҮәе“Әдәӣи®°еҪ•жҳҜиҝҷдәӣйҮҚеҗҜзӮ№зҡ„гҖӮ

log ж–Үд»¶еҶ…жҳҜ key ж— еәҸзҡ„пјҢиҖҢ Memtable дёӯжҳҜ key жңүеәҸзҡ„гҖӮ

жҹҘжүҫи®°еҪ•

д»Һ Memtable еҲ° Immutable MemtableпјҢеҶҚд»Һ Immutable Memtable еҲ°ж–Үд»¶пјҢиҖҢж–Үд»¶жҳҜд»ҺдҪҺ level еҲ°й«ҳ level иҝҷд№ҲдёҖдёӘжҹҘиҜўи·Ҝеҫ„

level 0зҡ„ж–Үд»¶жҹҘжүҫпјҢеҗҢе…¶д»–levelж–Үд»¶зҡ„жҹҘжүҫдёҚеҗҢ(level 0еӯҳеңЁдёҚеҗҢж–Үд»¶йҮҚеҸ keyзҡ„жғ…еҶөпјҢйңҖиҰҒжҢүж–°йІңеәҰиҝӣиЎҢжҺ’еәҸ)
